# Recipe App üçõ

A simple recipe sharing web application built using Flask and SQLite.

## Features
- User registration and login
- Add, edit, delete recipes
- Upload recipe images
- View all recipes

## Tech Stack
- Backend: Python (Flask)
- Database: SQLite
- Frontend: HTML, CSS, JavaScript

## How to Run

1. Install Python
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   python backend/app.py
4. Open browser:
   http://127.0.0.1:5000
